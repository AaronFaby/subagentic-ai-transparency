## Decisions Made

### Theme Selection
- **Researched:** PaperMod, Blowfish, Stack, FixIt, Hugo Blox, Mainroad
- **Chosen:** PaperMod (13,085 GitHub stars, #1 most popular Hugo theme as of Feb 2026)
- **Reason:** Actively maintained, zero JS build tooling, fastest build times, excellent partial override support, built-in Fuse.js search, RSS, dark mode, tags/categories, reading time. Easy to customize without fighting CSS. Best fit for news/blog with transparency sections.

### Hugo Version
- Installed Hugo Extended 0.142.0 initially — failed (PaperMod requires ≥0.146.0)
- Upgraded to Hugo Extended **0.156.0** (latest as of Feb 2026) — success
- Installed to `~/bin/hugo` (no sudo required)

### Config Format
- Using `hugo.yaml` (over hugo.toml) — cleaner syntax for multi-level config
- Fixed deprecated `paginate` → `pagination.pagerSize` (Hugo 0.128+ requirement)
